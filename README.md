# GSoC-ML4SCI-Miguel-Pamanes
Tasks for application to Google Summer of Code. For Machine Learning for Science.


In this repository, you will find the work I developed as part of my application to Google Summer of Code (GSoC).

I chose to apply to the Machine Learning for Science (ML4Sci) organization because it represents a unique opportunity to contribute to impactful research at the intersection of physics and machine learning. My long-term goal is to pursue a career in quantum computing, and I see this experience as a valuable step toward building both the technical skills and research perspective required in this field.

Through this project, I aim not only to strengthen my knowledge, but also to actively contribute to the development of innovative tools for scientific discovery.
